This is the official doc file for deploying a web application using heroku.
Find more information about heroku at:
    https://www.heroku.com

Heroku is a cloud service platform for applications for managing server files, deployment and other application based 
purposes. It supports many programming languages.

Heroku will be installed via the command line using zsh commands (mac & linux systems)
You can use PuTTY to SSH into the server you are using and run linux-like commands
using the PuTTy client. If you are using Windows OS this is an easy application
to get started with.

    See Installing and Using PuTTY : 
        Download the documentation here
        https://www.chiark.greenend.org.uk/~sgtatham/putty/docs.html
